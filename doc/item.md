# アイテム

- **戻** : 使用後所定の位置へ戻る.
- **捨** : 捨札として使用.使用後所定の位置へ戻る.魔理沙は特殊能力により,捨札にできない
- **自** : 自分の手番「特殊能力の使用」の行動として使用できる
- 特に書いていないものはいつでも使用可能.
- ⚠ : アイテムは必ず使用宣言を行なう
  - 例: フランの特殊能力で「相手が使おうとしたアイテムを破壊する」場合(呪法書など)その宣言に応じるため.
  - 例 : 解毒剤を持ってるから飲み過ぎ無効、と宣言する
